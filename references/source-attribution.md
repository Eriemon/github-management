# Source Attribution

This skill includes migrated behavior and reference material derived from the OpenAI curated GitHub and security skills that were used as development references:

- `gh-address-comments`: PR comment and review-thread collection workflow.
- `gh-fix-ci`: PR check inspection, GitHub Actions log extraction, and external CI boundary handling.
- `security-best-practices`: language and framework security review guidance.
- `security-threat-model`: repository-grounded threat modeling prompt and controls guidance.
- `security-ownership-map`: git-history ownership analysis scripts and Neo4j import guidance.

The migrated files live in standard skill folders only: `scripts/` and `references/`. The development cache directory was intentionally removed so runtime use depends only on standard SKILL resources.
